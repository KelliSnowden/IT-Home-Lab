# Windows 11 Networking Lab

## Overview
This lab was performed inside a Windows 11 virtual machine running on QEMU/KVM to practice foundational networking and troubleshooting skills.

## Commands Practiced
- ipconfig
- ping
- nslookup
- tracert

## Skills Learned
- DNS troubleshooting
- ICMP connectivity testing
- Network path tracing
- Virtual machine networking
- Windows command-line navigation

## Tools Used
- Windows 11 VM
- QEMU/KVM
- Command Prompt

## Notes
Sensitive information such as internal IP addresses was sanitized before upload as part of cybersecurity best practices.
