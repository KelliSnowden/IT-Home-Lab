# 🦈 Wireshark Packet Analysis Lab

## 📋 Overview

This lab focuses on capturing and analyzing network traffic using Wireshark.

The goal is to gain hands-on experience with packet analysis, network protocols, filtering traffic, and troubleshooting network communications.

---

## 🎯 Objectives

- Capture network traffic
- Analyze ICMP packets
- Analyze DNS traffic
- Practice Wireshark filters
- Understand packet flow and network communication

---

## 🛠️ Tools Used

| Tool | Purpose |
|---|---|
| Windows 11 VM | Lab environment |
| Wireshark | Packet capture and analysis |
| Command Prompt | Generate network traffic |

---

## 🔍 Activities Performed

### ICMP Analysis

Captured traffic generated by the `ping` command.

### DNS Analysis

Captured DNS requests and responses generated by `nslookup`.

### Traffic Filtering

Used Wireshark display filters to isolate specific traffic types.
